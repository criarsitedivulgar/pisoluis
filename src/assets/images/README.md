## Como Substituir as Imagens

1. Substitua os arquivos nesta pasta pelas suas próprias fotos
2. Mantenha os mesmos nomes de arquivo
3. Recomendações de tamanho:
   - Logo: 400x400px (PNG com fundo transparente)
   - Hero: 1920x1080px
   - Sobre Nós: 800x600px
   - Serviços: 1200x800px
   - Portfolio: 1200x800px

## Formatos Recomendados
- Logo: PNG (com fundo transparente)
- Fotos: JPG ou WebP
- Vídeos: MP4

## Dica
Para melhor performance, otimize suas imagens antes de adicionar ao site usando ferramentas como TinyPNG ou Squoosh.
